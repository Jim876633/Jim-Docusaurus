【Mac】常用快捷鍵

剛開始用 Mac 最困惑的就是 command 跟 opetion ，可以把他當成是 Windows 的 ctrl 跟 alt，大部分的快捷鍵照著這個邏輯 90% 都可以通用，我把快捷分成基本功能跟進階功能，基本的就是像複製貼上這種大部分人會用到的，進階的就是一些螢幕關閉縮小等等，常用滑鼠操作的，就算不知道也不影響的功能。

基本功能​

功能MacWindows
複製command + cctrl + c
貼上command + Vctrl + V
剪下command + Xctrl + X
復原command + Zctrl + Z
儲存command + Sctrl + S
選取所有項目command + Actrl + A
尋找頁面項目command + Fctrl + F
將字體變粗command + Fctrl + F

進階功能​

功能MacWindows
關閉當前視窗command + Wctrl + W
關閉應用程式
（等於按下叉叉）		command + Qctrl + Shift + W
or Alt + F4
縮小目前視窗
（在全螢幕模式無法使用）		command + M先按 Alt + Space 再按 N
開啟新視窗command + Nctrl + N
開啟無痕視窗command + Shift + Nctrl + Shift + N
全螢幕（退出全螢幕）command + ctrl + F
or Fn + F		F11
切換 app 視窗command + tabalt + tab
切換輸入法中/英 or ctrl + SpaceShift
Spotlight 搜尋
（ Mac 才有 ）		command + Space
全螢幕截圖command + Shift + 3Prtsc
自行選取範圍截圖command + Shift + 4Windows + Shift + S
（開啟剪取工具）
顯示 finder 隱藏資料command + Shift + .
