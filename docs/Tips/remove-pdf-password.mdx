有時候下載一些銀行存摺 PDF，會有密碼設定，但是要傳給別人時，就不想要設置密碼權限，這一篇就來告訴你一個簡單的方法移除權限。

## 使用 Chrome 列印功能

下載完 PDF 後，使用 Chrome 打開，可以直接把 PDF 拉到 Chrome 頁面，就可以打開了。

這時候還會需要輸入密碼
