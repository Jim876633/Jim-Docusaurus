## Step1

在搜尋的地方選擇 Use Regular Expression，並輸入

```
console.log.*$
```

<Img src={image_1} />

## Step2

可以選擇 replace all 把所有 console.log() 轉成空字串
或是只選擇想刪除的 console.log()
