在使用雙螢幕的時候，有時候會想要將應用程式視窗移動到另一個螢幕上，大部分的人可能都是直接用滑鼠拖曳，這一篇就來介紹一個更快速的方法。

## Raycast

[Raycast](https://raycast.com/) 是一個 macOS 上的快速啟動工具，可以用快捷鍵呼叫出搜尋列，並且可以透過安裝不同的插件來擴充功能。

下載後就有內建的 `Window Management` 插件，可以用來移動視窗。

<Img src={image_1} />

有常見的移動視窗到螢幕的左右上下，對於分割螢幕非常方便。

而我們要使用的移動視窗到另一個螢幕的功能，則是在插件的 `Next Display`。

<Img src={image_2} />

可以看到我這邊設定的快捷鍵是 `⇧ + ⌘ + 空格`，這樣就可以快速的在不同螢幕裝置間移動視窗了！
