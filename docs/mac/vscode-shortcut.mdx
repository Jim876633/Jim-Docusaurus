## 基本功能

|      功能      |            Mac             |        Windows         |
| :------------: | :------------------------: | :--------------------: |
|    整行刪除    |    `command` + `delete`    | `ctrl` + `shift` + `k` |
|   單文字刪除   |    `option` + `delete`     |    `alt` + `Delete`    |
| 折行 word wrap |       `option` + `z`       |      `alt` + `z`       |
| 移動單行程式碼 |   `option` + `↑` or `↓`    |   `alt` + `↑` or `↓`   |
|    另開分頁    |      `command` + `\`       |                        |
|    切換分頁    | `command`+`k` + `→` or `←` |                        |
