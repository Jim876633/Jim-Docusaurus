剛開始用 Mac 最困惑的就是 `command` 跟 `opetion` ，可以把他當成是 Windows 的 `ctrl` 跟 `alt`，大部分的快捷鍵照著這個邏輯 90% 都可以通用，我把快捷分成基本功能跟進階功能，基本的就是像複製貼上這種大部分人會用到的，進階的就是一些螢幕關閉縮小等等，常用滑鼠操作的，就算不知道也不影響的功能。

## 基本功能

|     功能     |       Mac       |   Windows    |
| :----------: | :-------------: | :----------: |
|     複製     | `command` + `c` | `ctrl` + `c` |
|     貼上     | `command` + `V` | `ctrl` + `V` |
|     剪下     | `command` + `X` | `ctrl` + `X` |
|     復原     | `command` + `Z` | `ctrl` + `Z` |
|     儲存     | `command` + `S` | `ctrl` + `S` |
| 選取所有項目 | `command` + `A` | `ctrl` + `A` |
| 尋找頁面項目 | `command` + `F` | `ctrl` + `F` |
|  將字體變粗  | `command` + `F` | `ctrl` + `F` |

## 進階功能

|                   功能                    |                     Mac                     |                     Windows                      |
| :---------------------------------------: | :-----------------------------------------: | :----------------------------------------------: |
|               關閉當前視窗                |               `command` + `W`               |                   `ctrl` + `W`                   |
|    關閉應用程式 <br/>（等於按下叉叉）     |               `command` + `Q`               |   `ctrl` + `Shift` + `W`<br/> or `Alt` + `F4`    |
| 縮小目前視窗<br/>（在全螢幕模式無法使用） |               `command` + `M`               |          先按 `Alt` + `Space` 再按 `N`           |
|                開啟新視窗                 |               `command` + `N`               |                   `ctrl` + `N`                   |
|               開啟無痕視窗                |          `command` + `Shift` + `N`          |              `ctrl` + `Shift` + `N`              |
|           全螢幕（退出全螢幕）            | `command` + `ctrl` + `F`<br/> or `Fn` + `F` |                      `F11`                       |
|               切換 app 視窗               |              `command` + `tab`              |                  `alt` + `tab`                   |
|                切換輸入法                 |         `中/英` or `ctrl` + `Space`         |                     `Shift`                      |
|    Spotlight 搜尋 <br/>（ Mac 才有 ）     |             `command` + `Space`             |                                                  |
|                全螢幕截圖                 |          `command` + `Shift` + `3`          |                     `Prtsc`                      |
|             自行選取範圍截圖              |          `command` + `Shift` + `4`          | `Windows` + `Shift ` + `S` <br/>（開啟剪取工具） |
|           顯示 finder 隱藏資料            |          `command` + `Shift` + `.`          |                                                  |
